## 创建Vue实例

每个 Vue 应用都是通过用 `Vue` 函数创建一个新的 **Vue 实例**开始的：

```
var vm = new Vue({
  el: '#app',
  data: function () {
    return {
      
    }
  },
  mounted: function () {
    
  },
  methods: {
    // 钩子函数
  }
})
```
